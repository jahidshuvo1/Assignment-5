Question-2: What is the difference between innerHTML, innerText, and textContent?

............................................................................................................................................................................................................................................................................................................................................

In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements.

Question-3: What is event delegation in the DOM?

............................................................................................................................................................................................................................................................................................................................................

Event delegation is a technique in JavaScript where a parent element handles events for its child elements, leveraging event bubbling to efficiently manage events, especially for dynamically added or removed elements.

Question-4: What is event bubbling in the DOM?

............................................................................................................................................................................................................................................................................................................................................

In the DOM, event bubbling is a mechanism where an event triggered on a child element propagates upwards through its parent and ancestor elements, allowing parent elements to respond to events originating from their children.
